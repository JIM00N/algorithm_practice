# Stack

스택은 자료구조의 일종이다.

stack, 말 그대로 쌓는 모습을 생각하면 된다.

처음 들어간 것이 가장 마지막에 나올 수 있고, 마지막에 들어간 것이 처음 나올 수 있다.

이러한 특징을 '후입선출', Last In First Out(LIFO),이라 한다.

요소의 입력과 출력은 'push'와 'pop' 이라 부른다.

pop은 스택에 마지막 요소를 출력하고 스택에서 그 요소를 삭제한다.
